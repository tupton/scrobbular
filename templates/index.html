What is Scrobbular?

Scrobbular makes it easy to scrobble songs to Last.fm from anywhere.

Once you allow access to Last.fm, telling the world about what music you listen to is as simple as sending a message to Scrobbular. With curl, that would look something like this:

curl -X POST --data-urlencode "username=<Your Google Username>" \
    --data-urlencode "s=<Your Scrobbular Secret>" \
    --data-urlencode "track=Red Sky" \
    --data-urlencode "artist=Thrice" \
    --data-urlencode "album=Vheissu" \
    --data-urlencode "duration=258" \
    http://scrobbular.appspot.com/scrobble

How do I start using Scrobbular?

You can start using Scrobbular with your Google Account.

First, you need to allow Scrobbular to talk to Last.fm on your behalf. You can manage your user page here once you log in with a Google Account. That's it! You can start using Scrobbular by sending simple messages without the need to figure out how to use Last.fm's API.
